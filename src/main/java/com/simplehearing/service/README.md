# Service Package

This package contains service classes that implement business logic.

## Services

- **AppointmentService.java**: Business logic for appointment management
- **AvailabilityService.java**: Logic for checking service availability
- **BlogService.java**: Blog post management logic
- **ContactService.java**: Contact message processing
- **GalleryService.java**: Gallery item management
- **HearingServiceService.java**: Service catalog management
- **PaymentService.java**: Payment processing logic

## Features

- Transaction management with @Transactional
- Input validation and business rule enforcement
- Integration with external services (payment gateways)
- Data transformation between entities and DTOs
- Email notifications and other side effects
