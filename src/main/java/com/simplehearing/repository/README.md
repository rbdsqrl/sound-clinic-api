# Repository Package

This package contains Spring Data JPA repositories for database operations.

## Repositories

- **AppointmentRepository.java**: Database operations for appointments
- **BlogPostRepository.java**: Operations for blog posts
- **ContactMessageRepository.java**: Operations for contact messages
- **GalleryItemRepository.java**: Operations for gallery items
- **HearingServiceRepository.java**: Operations for hearing services
- **PaymentRepository.java**: Operations for payments

## Features

- Automatic query generation from method names
- Custom query methods using @Query
- Pagination and sorting support
- Relationship fetching with @EntityGraph
- Transaction management
