# Request DTOs

This directory contains Data Transfer Objects for API request payloads.

## Common DTOs

- **AppointmentRequest.java**: Data for creating/updating appointments
- **BlogPostRequest.java**: Data for creating blog posts
- **ContactMessageRequest.java**: Data for contact form submissions
- **GalleryItemRequest.java**: Data for adding gallery items
- **HearingServiceRequest.java**: Data for creating services
- **PaymentRequest.java**: Data for payment operations

## Features

- Input validation using Bean Validation annotations (@NotNull, @Size, etc.)
- Proper data types for API contracts
- Clear naming conventions
