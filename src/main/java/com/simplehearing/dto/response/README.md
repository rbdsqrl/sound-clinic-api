# Response DTOs

This directory contains Data Transfer Objects for API response payloads.

## Common DTOs

- **AppointmentResponse.java**: Appointment data returned to clients
- **BlogPostResponse.java**: Blog post data for responses
- **ContactMessageResponse.java**: Contact message data
- **GalleryItemResponse.java**: Gallery item data
- **HearingServiceResponse.java**: Service catalog data
- **PaymentResponse.java**: Payment status and details

## Features

- Controlled data exposure (no sensitive internal fields)
- Consistent response structure across the API
- Support for pagination and metadata where applicable
