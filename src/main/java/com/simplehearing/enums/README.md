# Enums Package

This package contains enumeration classes used throughout the application.

## Enums

- **AppointmentStatus.java**: Status values for appointments (PENDING, CONFIRMED, CANCELLED, COMPLETED)
- **GalleryType.java**: Types of gallery items (PHOTO, VIDEO)
- **PaymentStatus.java**: Payment transaction statuses (PENDING, PAID, FAILED, REFUNDED)

## Purpose

Enums provide:
- Type safety for status and type fields
- Consistent values across the application
- Database-level constraints
- Clear API documentation
